# REAL ESTATE API WITH GO, SQLITE AND GIN

## Getting Started

### Prerequisites

- Go 1.25+
- Gin
- SQLite

### Local Development

1. Clone the repository:

```bash
git clone https://github.com/Ademayowa/deploy-go-demo
cd deploy-go-demo
```

2. Install dependencies:

```bash
go mod download
```

3. Running the Project:

```bash
go run main.go
```
